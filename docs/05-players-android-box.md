# 05. Плеер на обычном телевизоре через Android-приставку

Если телевизор **обычный** (не Smart, только HDMI/USB) или его Smart-функции не
поддерживают приложение Xibo — подключаем к нему дешёвую **Android-ТВ-приставку**.
Приставка играет видео и выводит его на телевизор по HDMI.

## Что купить

Подойдёт почти любая Android TV box с характеристиками:

| Параметр | Минимум | Комфортно |
|----------|---------|-----------|
| ОС | Android 8+ (лучше Android TV) | Android 11+ |
| RAM | 2 ГБ | 4 ГБ |
| Память | 8 ГБ | 16–32 ГБ (кэш видео) |
| Видеовыход | HDMI | HDMI |
| Сеть | Wi-Fi | Wi-Fi + Ethernet-порт |

Ориентир по цене: ~2 000–4 000 ₽ за приставку. Для стабильности лучше брать
модели с **Ethernet-портом** и ставить, где можно, на провод.

> Альтернатива приставке: **Raspberry Pi** или мини-ПК на Windows. Приставка на
> Android обычно самый дешёвый и простой вариант. Если хотите полный контроль над
> софтом на точке и обновления «по воздуху» на весь флот — см. отдельный документ
> [13-custom-image.md](13-custom-image.md) (Raspberry Pi + ArexiboOS / balenaOS).
> Учтите: официального плеера Xibo для Raspberry Pi нет, там используется
> сообществный плеер — детали в том же документе.

---

## Шаг 1. Подключить приставку к телевизору

1. Вставьте приставку в **HDMI**-разъём телевизора.
2. Подключите питание приставки.
3. На пульте телевизора выберите нужный **вход HDMI** (HDMI1/HDMI2 — тот, куда
   воткнули).
4. На экране появится интерфейс Android — значит, всё соединено.

---

## Шаг 2. Подключить приставку к интернету

Настройки Android → Сеть → Wi-Fi (ввести пароль) **или** воткнуть кабель
Ethernet. Проверьте, что интернет есть.

---

## Шаг 3. Установить и настроить Xibo Player

Дальше — **точно так же, как для Smart TV**:

1. Google Play → установить **Xibo Player**.
2. В настройках плеера указать:
   - **CMS Address** — адрес вашего сервера;
   - **Key** — CMS Secret Key;
   - **Display Name** — понятное имя, напр. `Клиника-Север — стоматология`.
3. В веб-панели **Displays** авторизовать новый экран.

Полностью эти шаги расписаны в [04-players-smart-tv.md](04-players-smart-tv.md),
шаги 3–5 — они идентичны.

---

## Шаг 4. Сделать так, чтобы всё поднималось само после отключения питания

Это критично для точек, где никто не будет «оживлять» приставку вручную.

1. **Автозапуск приставки при подаче питания.** Многие Android-приставки
   включаются сами при появлении питания. Если нет — проверьте настройки
   питания/загрузки в прошивке.
2. **Автозапуск Xibo Player при старте.** В настройках приложения Xibo включите
   опцию запуска при загрузке, либо используйте утилиту автозапуска. Цель — после
   выключения/включения розетки приставка сама открыла плеер и продолжила показ.
3. **Телевизор должен сам включаться и выбирать нужный HDMI.** У части ТВ есть
   функция «включаться при HDMI-сигнале» (HDMI-CEC / «включение по сигналу»).
   Включите её, чтобы телевизор просыпался вместе с приставкой.
4. Отключите на приставке спящий режим и заставку.

**Проверка:** выдерните питание всей точки (ТВ + приставка), воткните обратно.
Через 1–2 минуты на экране должно само появиться видео, без единого нажатия.

## Практические советы

- **Подпишите приставки.** Наклейте на каждую стикер с именем точки — совпадающим
  с Display Name в панели. Сильно упростит поддержку.
- **Ставьте на провод, где можно.** Wi-Fi в клинике может быть перегружен;
  Ethernet надёжнее для видео.
- **Единая модель приставки.** Закупите одинаковые приставки — их проще
  настраивать и заменять по единой инструкции.
