# 12. Словарь терминов Xibo

Короткий перевод «терминов из панели» на человеческий язык.

| Термин (в панели) | По-русски | Что это |
|-------------------|-----------|---------|
| **CMS** | Сервер управления | «Мозг» системы: веб-панель + база + хранилище видео |
| **Display** | Экран | Один зарегистрированный плеер = один телевизор |
| **Display Group** | Группа экранов | Набор экранов, которым удобно назначать контент разом |
| **Player** | Плеер | Приложение Xibo на устройстве (Smart TV, приставка, ПК) |
| **Media** | Медиа | Загруженный файл: видео, картинка, аудио |
| **Library** | Библиотека | Хранилище всех загруженных медиа |
| **Layout** | Раскладка | Что видно на экране: регионы с видео/картинками/текстом |
| **Region** | Регион | Прямоугольная область на раскладке, где играет контент |
| **Playlist** | Плейлист | Список медиа, играющих по очереди внутри региона |
| **Dynamic Playlist** | Умный плейлист | Плейлист, который сам включает все медиа с заданным тегом |
| **Campaign** | Кампания | Несколько раскладок, объединённых для показа по очереди |
| **Schedule** | Расписание | Правила «что, кому и когда показывать» |
| **Event** | Событие | Одна запись в расписании (раскладка → группа → время → приоритет) |
| **Priority** | Приоритет | Чем выше, тем «главнее» раскладка на экране (перекрывает низшие) |
| **Publish** | Опубликовать | Сделать черновик раскладки «боевым», доступным плеерам |
| **Collect interval** | Интервал опроса | Как часто плеер спрашивает сервер об обновлениях |
| **Collect Now** | Обновить сейчас | Команда плееру немедленно забрать обновления |
| **CMS Secret Key** | Ключ регистрации | Пароль, которым плеер «представляется» серверу при первом подключении |
| **Authorise / License** | Авторизация экрана | Разрешение конкретному плееру подключаться к CMS |
| **XMR** | Служба быстрых команд | Позволяет серверу мгновенно «толкать» команды плеерам (порт 9505) |
| **Tag** | Тег | Метка на медиа/раскладке для поиска и умных плейлистов |
| **Transition** | Переход | Анимация смены контента |

Полный официальный глоссарий и руководство: https://xibosignage.com/manual
