# 09. Ежедневное использование (шпаргалка)

Короткие пошаговые рецепты на типовые задачи. Держите этот файл под рукой.

---

## 🔄 Поменять видео сразу на всех телевизорах

1. **Library** → загрузите новый ролик (см. [07](07-content-and-playlists.md)).
2. Откройте **общую раскладку** (напр. `Общее — июль`) → **Layouts** → Edit.
3. Замените видео в регионе на новое (удалить старое, добавить новое) →
   **Publish**.
4. Готово. Экраны группы `Все` подтянут обновление при следующем опросе (обычно
   в течение нескольких минут).

**Хотите мгновенно?** В **Displays** выделите экраны → **Collect Now** (сервер
через XMR попросит плееры обновиться сейчас же).

> Вариант без правки раскладки: если раскладка ссылается на плейлист, можно просто
> поменять содержимое плейлиста — раскладку публиковать заново не нужно.

---

## 👶 Поменять видео только в детских отделениях

1. Загрузите детский ролик в **Library**.
2. Откройте **детскую раскладку** (напр. `Детское — июль`) → замените видео →
   **Publish**.
3. Обновится только у группы `Детские`. Остальных не коснётся.

---

## ➕ Добавить новую точку (телевизор)

1. Настройте плеер:
   - Smart TV → [04-players-smart-tv.md](04-players-smart-tv.md);
   - обычный ТВ + приставка → [05-players-android-box.md](05-players-android-box.md).
2. В **Displays** авторизуйте новый экран (License → разрешён).
3. Добавьте экран в группу **`Все`** (и в спец-группу, если это, напр., детское).
4. Всё. Контент прилетит автоматически, отдельно назначать не нужно.

---

## 🏥 Дать одной клинике особый контент

1. Создайте группу под клинику (если ещё нет): **Display Groups → Add**, напр.
   `Клиника-Север`.
2. Добавьте экраны этой клиники в группу.
3. **Schedule → Add Event**: нужная раскладка → эта группа → приоритет выше 0
   (напр. 20).

---

## 📢 Временное объявление (акция, режим работы) на период

1. Сделайте раскладку с объявлением.
2. **Schedule → Add Event**: выберите группу (всю сеть или конкретную клинику),
   задайте **даты начала и конца**, приоритет выше обычного (напр. 50).
3. По окончании периода экраны сами вернутся к обычному контенту — вручную
   выключать не надо.

---

## 🧪 Правильно раскатывать изменения (чтобы не сломать всё сразу)

1. Сначала назначьте новую раскладку на группу **`Тест`** (1 экран).
2. Убедитесь, что играет корректно.
3. Затем меняйте общую/детскую раскладку для боевых групп.

---

## 👀 Проверить, что происходит на точках

- **Displays** — список экранов: онлайн/офлайн, когда последний раз выходили на
  связь, что показывают.
- Красный/серый статус = экран давно не отвечает → см.
  [11-troubleshooting.md](11-troubleshooting.md).
- **Display Groups → Members** — быстро увидеть, кто в какой группе.

---

## 📋 Мини-чек-лист «обновление контента»

- [ ] Новый файл пережат в MP4/H.264 1080p
- [ ] Загружен в Library, назван понятно, отмечен тегом
- [ ] Проверен на группе `Тест`
- [ ] Обновлена нужная раскладка (общая / детская / клиника) и **Publish**
- [ ] (Опц.) Collect Now для мгновенного применения
- [ ] Проверено на 1–2 реальных экранах, что играет
- [ ] Флешечные точки (если есть) обновлены по регламенту
      [06-usb-fallback.md](06-usb-fallback.md)
